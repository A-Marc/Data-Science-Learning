## 🔢 NumPy Fundamentals

> *"NumPy is the foundation of the Python data science ecosystem."*

<div align="center">

### 📂 Folder: `004 NumPy/`
### ⚡ **High-Performance Numerical Computing**

</div>

Master the core library for numerical computing in Python. NumPy provides powerful N-dimensional arrays, mathematical functions, and tools that are **50-100x faster** than pure Python for numerical operations.

### 🎯 Why NumPy?

```python
# Python List vs NumPy Array Performance
import time
import numpy as np

# Python List (Slow)
list1 = list(range(10000))
list2 = list(range(10000))
start = time.time()
result = [x + y for x, y in zip(list1, list2)]
print(f"Python List: {time.time() - start:.4f}s")

# NumPy Array (Fast!)
arr1 = np.array(list1)
arr2 = np.array(list2)
start = time.time()
result = arr1 + arr2
print(f"NumPy Array: {time.time() - start:.4f}s")
# NumPy is 50-100x faster! ⚡
```

SELECT department_id, AVG(salary) as avg_salary
FROM employees
GROUP BY department_id
HAVING AVG(salary) > 60000;

-- Complex aggregation
SELECT 
    d.department_name,
    COUNT(e.id) as num_employees,
    AVG(e.salary) as avg_salary,
    MAX(e.salary) as max_salary
FROM departments d
LEFT JOIN employees e ON d.id = e.department_id
GROUP BY d.department_name
ORDER BY avg_salary DESC;

| Concept | Description | Example |
|---------|-------------|---------|
| 🎲 **Sample Space** | Set of all possible outcomes | Rolling a die: {1, 2, 3, 4, 5, 6} |
| 🎯 **Event** | Subset of sample space | Getting an even number: {2, 4, 6} |
| 📊 **Probability** | P(E) = favorable / total outcomes | P(even) = 3/6 = 0.5 |
| 🔗 **Conditional** | P(A\|B) = P(A ∩ B) / P(B) | Probability of rain given clouds |

| 🎭 **Bayes' Theorem** | P(A\|B) = P(B\|A) × P(A) / P(B) | Medical diagnosis, spam filtering |

### 🧮 Essential Formulas

```
Basic Probability:
P(A) = Number of favorable outcomes / Total outcomes

Addition Rule:
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

Multiplication Rule:
P(A ∩ B) = P(A) × P(B|A)

Bayes' Theorem:
P(A|B) = P(B|A) × P(A) / P(B)

Expected Value:
E(X) = Σ [x × P(x)]
```
