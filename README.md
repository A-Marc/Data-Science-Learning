## 🚀 Quick Start

### 1. Clone Repository

```bash
git clone https://github.com/ggauravky/Data-Science-Learning.git
cd Data-Science-Learning
```

### 2. Setup Environment

```bash
# Using pip
pip install numpy pandas matplotlib seaborn jupyter beautifulsoup4 requests

# OR using conda (recommended)
conda create -n datasci python=3.11 -y
conda activate datasci
conda install numpy pandas matplotlib seaborn jupyter -y
pip install beautifulsoup4 requests
```

### 3. Launch Jupyter

```bash
jupyter notebook
```

### 4. Start Learning!

Navigate to `002 Python refresher/` and open `01_python_basic.ipynb` 🎉

```plaintext
  └─ Numerical computing basics

  └─ Coders of Delhi recommendation system
```

### 🌿 Intermediate (Weeks 5-8)

```plaintext
Week 5-6: Data Visualization
  └─ Matplotlib & Seaborn
  └─ Create professional charts

Week 7: Web Scraping
  └─ Extract data from websites
  └─ Build datasets

Week 8: SQL Databases
  └─ Database design
  └─ Complex queries
```

### 🌳 Advanced (Weeks 9-12)

```plaintext
Week 9-10: Advanced SQL
  └─ Joins, views, stored procedures

  └─ Probability theory
```
