## Python Refresher

A comprehensive review of Python fundamentals for data science, with detailed notes and examples in Jupyter notebooks and PDFs.

**Topics Covered:**
- **Python Basics:** Variables, Data Types, and Typecasting.
- **Strings:** Indexing, Slicing, and String Methods.
- **Operators:** Arithmetic, Comparison, Logical, and more.
- **Conditional Statements:** `if`, `elif`, `else`.
- **Loops:** `for` and `while` loops.
- **Data Structures:**
    - **Lists:** Mutable, ordered collections.
    - **Tuples:** Immutable, ordered collections.
    - **Sets:** Unordered, unique collections.
    - **Dictionaries:** Key-value pairs.
- **File Handling:** Reading from and writing to files.
- **JSON Module:** Working with JSON data.
- **Object-Oriented Programming (OOP):** Classes, objects, inheritance, and polymorphism.
- **Advanced Topics:** List comprehensions and lambda functions.

## Project: Coders of Delhi

A hands-on project that applies Python skills to solve a real-world problem. This project focuses on building a simple recommendation system.

**Project Files:**
- `data.json`, `massive_data.json`, `codebook_data.json`: Sample datasets for the project.
- `data_read.ipynb`: Notebook for reading and cleaning the data.
- `people_you_may_know.ipynb`: A recommendation algorithm to suggest friends.
- `pages_you_might_like.ipynb`: A recommendation algorithm for pages.
- `cleaned_codebook_data.json`: The cleaned dataset after pre-processing.

## NumPy Essentials

Learn the fundamentals of NumPy, the core library for numerical computing in Python.

**Topics Covered:**
- **NumPy Basics:** Creating arrays, understanding `ndarray` objects.
- **Array Creation:** From lists, using `zeros`, `ones`, `arange`, and `linspace`.
- **Array Attributes:** `shape`, `size`, `ndim`, `dtype`.
- **Data Types:** Working with different NumPy data types.
- **Indexing and Slicing:** Accessing and modifying array elements.
- **Broadcasting and Vectorization:** Performing efficient operations on arrays.

2.  **Navigate to the desired folder:**
