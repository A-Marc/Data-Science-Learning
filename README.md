## 🚀 How to Use This Repository

### 1. **Clone the Repository**
```bash
git clone https://github.com/ggauravky/Data-Science-Learning.git
cd Data-Science-Learning
```

### 2. **Set Up Your Environment**
```bash
# Create a conda environment
conda create -n datasci python=3.9
conda activate datasci

# Install required packages
pip install numpy pandas matplotlib seaborn jupyter requests beautifulsoup4
```

### 3. **Navigate and Explore**
```bash
# Explore different modules
cd "002 Python refresher"     # Start with Python basics
cd "004 NumPy"               # Move to NumPy tutorials
cd "005 Pandas"              # Learn data manipulation
# ... and so on
```

### 4. **Open Jupyter Notebooks**
```bash
jupyter notebook
# or
jupyter lab
```

### 5. **Follow the Learning Path**
1. **Start with Python Refresher** (if needed)
2. **Master NumPy** for numerical computing
3. **Learn Pandas** for data manipulation
4. **Practice with the Coders of Delhi Project**
5. **Explore Data Visualization**
6. **Dive into Web Scraping**
